# # 📁 Python File Transfer Lab (Educational)

> **Educational Project for Cybersecurity Students**

This project demonstrates the fundamentals of **TCP socket programming** and **file transfer** using Python in a **controlled lab environment**.

## ⚠️ Disclaimer

This project is provided **strictly for educational purposes**.

* Use only on devices and networks that **you own or have explicit permission to test**.
* Do **not** use this project to access, monitor, or transfer files from other people's systems.
* The author is **not responsible** for any misuse of this project.

* Python Socket Programming
* TCP Client-Server Communication

* Network Programming Basics
